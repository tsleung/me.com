how to play
  1. Pick a bit each round: 0 or 1 (keys work too).
  2. The machine guesses your bit before you click. If it matches, the machine scores; if not, you do.
  3. Try to stay unpredictable. The only way to beat it long-term is to play like a fair coin.
  4. Flip to lift the hood to watch the 8-cell memory update round by round.

The original machine was built by Claude Shannon at Bell Labs in 1953 as a riff on Dave Hagelbarger’s SEER. It plays matching pennies against a human and wins far more than half the time because humans aren’t random.

how it works

Each round the machine forms a 3-bit key from recent outcomes and reads that memory cell. The cell stores a tendency — do you stay on the same bit or switch — plus a confidence flag that only flips after two disagreements. With enough rounds, any rhythm in your choices leaks through the bits.
